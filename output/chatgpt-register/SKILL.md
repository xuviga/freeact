# chatgpt-register

Automated ChatGPT registration via temporary email (emailtick.com).

## Quick Start

```bash
cd output/chatgpt-register
python skill.py
```

## How it works

1. Opens emailtick.com and extracts a temporary Gmail address
2. Navigates to ChatGPT login/signup page
3. Enters the temporary email
4. Clicks Continue
5. Returns to emailtick to check for verification code
6. Enters the verification code on ChatGPT
